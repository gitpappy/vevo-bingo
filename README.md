# Vevo Bingo AKA Music Video Drinking Game

Drinking game for Vevo TV. 2 - 4 players.

## Rules

* Take a sip when someone makes a match.
* Finish your drink when someone wins.

## Recommended drinks

* Mojito
* Champagne
* Rum and Coke
